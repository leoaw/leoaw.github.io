Frequently Asked Questions

Apa itu website?

Website adalah satu atau beberapa kumpulan halaman web yang saling berhubungan untuk memperkenalkan, menjual, dan mengiklankan produk, jasa, dan perusahaan Anda secara nasional maupun global.

Mengapa saya harus memiliki website?

Saat ini memiliki website adalah suatu hal yang sangat penting untuk memperkenalkan, menjual, dan mengiklankan produk, jasa, dan perusahaan Anda. Sama pentingnya dengan memiliki kantor/toko fisik atau nomor telepon. Dan pada saat ini juga hampir setiap produk, jasa, maupun perusahaan sudah memiliki website karena dengan memiliki website informasi produk, jasa, maupun perusahaan dapat diakses pelanggan Anda, 24 jam. Dengan memiliki website, Anda dapat meningkatkan penjualan produk atau jasa. Selain itu juga menghemat pengeluaran biaya seperti biaya sewa kantor atau toko, biaya cetak katalog, brosur, spanduk, dll.

Dapatkah saya melihat perkembangan website saat itu sedang dibuat?

Bisa. Saya akan menyediakan Anda dengan alamat web pribadi dengan sub-domain, di mana Anda dapat melihat perkembangan pengerjaan website Anda.

Apakah saya bisa mengedit halaman?

Bisa. Untuk pergantiannya bisa menghubungi saya.

Saya ingin memesan website, tetapi saya bingung menentukan desain?

Silahkan hubungi saya dan ceritakan kebutuhan Anda. Saya akan membantu memberikan beberapa referensi tampilan website yang sekiranya sesuai dengan kebutuhan Anda.

Apakah melayani pembuatan website memakai template website langsung jadi?

Pada dasarnya saya membuat website dengan desain kustom untuk bisa menyesuaikan keinginan klien yang berbeda-beda kebutuhannya. Namun jika Anda ingin menggunakan template langsung jadi, saya akan membantu.

Bagaimana mekanisme pembayaran jasa pembuatan website?

Untuk pembayaran di awal sebesar 40%, dan pelunasan sisanya setelah website telah selesai dikerjakan.

Berapa lama proses pembuatan website sampai siap digunakan?

Proses pembuatan website tergantung dari kesiapan data-data untuk website Anda. Semakin cepat dan lengkap Anda mengirimkan data-datanya, maka semakin cepat pula saya dalam mengerjakan website pesanan Anda. Namun pada umumnya proses pembuatan website memakan waktu paling cepat 1 minggu.

Apakah perlu melakukan pertemuan secara langsung sehubungan dengan pembuatan website?

Dalam sebagian besar kasus, pertemuan secara langsung tidak diperlukan. Namun untuk kelancaran pengerjaan, saya perlu berkomunikasi dengan Anda secara teratur untuk membahas desain website dan fungsi yang sesuai dengan yang Anda inginkan. Hal ini bisa dilakukan lewat Email/Telepon/WhatsApp/Telegram. Jika memang butuh untuk bertemu, maka saya akan dengan senang hati siap menemui Anda.
